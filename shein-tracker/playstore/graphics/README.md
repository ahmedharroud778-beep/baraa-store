# Graphics and Screenshots

This directory should contain promotional graphics and app screenshots.

## Required Graphics

### Feature Graphic
- **Size**: 1024x500 pixels
- **Format**: PNG or JPG
- **Purpose**: Featured image in Play Store listing

### Promo Image
- **Size**: 180x120 pixels
- **Format**: PNG or JPG
- **Purpose**: Small promotional image

## Required Screenshots

### Phone Screenshots
- **Minimum**: 2 screenshots
- **Recommended**: 4-8 screenshots
- **Size**: 1080x1920 or 1080x2400 (portrait)
- **Format**: PNG or JPG
- **Purpose**: Show app features in Play Store

## Screenshot Ideas

1. **Main Calculator Screen**
   - Show the price estimation interface

2. **Order Tracking Screen**

3. **Contact Screen**

4. **Settings/Options Screen**
   - Show app settings

5. **Offline Mode Screen**
   - Show offline functionality
   - Highlight reliability

## Creating Screenshots

### Using Android Emulator
1. Run the app on emulator
2. Navigate to desired screen
3. Take screenshot using Android Studio or device screenshot

### Using Real Device
1. Install the app
2. Navigate to desired screen
3. Take screenshot using device buttons

### Screenshot Tools
- Android Studio: Run > Take Screenshot
- ADB: `adb shell screencap -p /sdcard/screenshot.png`
- Online tools for adding device frames
