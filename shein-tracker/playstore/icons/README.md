# App Icons

This directory should contain the app icons in various sizes.

## Required Icons

### Launcher Icon (Adaptive)
- **Size**: 512x512 pixels
- **Format**: PNG with transparency
- **Purpose**: Main app icon on device home screen

### High-Resolution Icon
- **Size**: 1024x1024 pixels
- **Format**: PNG
- **Purpose**: Play Store listing

## Icon Guidelines

1. Use your brand colors and logo
2. Ensure good contrast and readability
3. Test on different backgrounds
4. Follow Android design guidelines
5. Avoid text in icons (except logo)

## Creating Icons

You can create icons using:
- Android Studio's Image Asset Studio
- Online tools like:
  - https://romannurik.github.io/AndroidAssetStudio/
  - https://makeappicon.com/
- Design software like Figma, Photoshop, etc.

## File Naming

- `icon_512.png` - 512x512 launcher icon
- `icon_1024.png` - 1024x1024 high-res icon
- `icon_foreground.png` - Adaptive icon foreground
- `icon_background.png` - Adaptive icon background
